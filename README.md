# 🧘 Yoga with Shuchita

A beautiful, responsive website for **Shuchita Parasher's** daily morning yoga classes.

## ✨ Features

- **Parallax scrolling** with layered backgrounds and smooth motion
- **Warm + cool tone palette** — amber, teal, and plum accents
- **CSS patterns & geometric art** — mandala, dot grids, sacred geometry (no stock photos)
- **Scroll reveal animations**, card tilt effects, and interactive gradient
- **Fully responsive** — works on desktop, tablet, and mobile
- **Google Fonts** — Cormorant Garamond + Inter

## 🗂️ Project Structure

```
yoga-shuchita/
├── index.html    # Main page
├── style.css     # Design system & styles
├── script.js     # Parallax, animations, interactivity
├── .gitignore
└── README.md
```

## 🚀 Usage

Simply open `index.html` in any modern browser — no build step required.

## 📜 About Bodhi School of Yoga

The Bodhi School of Yoga is a renowned yoga institution with studios across India (Hyderabad, Bengaluru, Vizag, Goa). They blend traditional yogic wisdom with modern science, and their programs are aligned with Yoga Alliance (USA) standards.
